## 什么时候调用什么

- **任务复杂** → 启用两级规划器，拆分子任务
- **需要回忆** → 调用记忆系统，搜索相关经验
- **需要专业知识** → 调用对应技能（代码/文档/数据分析）
- **需要操作界面** → 调用 GUI 技能（点击/输入/截图）
- **任务重复** → 沉淀到经验库，下次直接复用
- **需要多人视角** → 启动专家辩论，多角度分析
- **遇到错误** → 反思推理，换一种方法重试
