# Hello World Skill

A demonstration skill that prints hello world and performs basic file operations.

## Usage

This skill can be invoked via `cogu skills builtin` or loaded by the agent.

### Actions
- `greet`: Print a greeting message
- `count`: Count files in a directory
- `echo`: Echo back input text
