### CJK 字体映射

```javascript
// docx-js 中的 RunFonts 设置
run: {
  font: {
    ascii: "Calibri",       // 西文字符
    eastAsia: "SimSun",     // 中日韩字符
    hAnsi: "Calibri",       // 高 ANSI 字符
  },
}
```

## 表格设计

### 三线表（学术论文）

```javascript
const topBorder = { style: BorderStyle.SINGLE, size: 3, color: "000000" };
const headerBottom = { style: BorderStyle.SINGLE, size: 2, color: "000000" };
const bottomBorder = { style: BorderStyle.SINGLE, size: 3, color: "000000" };
const noBorder = { style: BorderStyle.NONE, size: 0, color: "FFFFFF" };

// 表头行: top + headerBottom
// 数据行: noBorder (top/bottom)
// 最后一行: bottomBorder
```

### 斑马纹（商务报告）

```javascript
// 奇数行
shading: { fill: "F2F2F2", type: ShadingType.CLEAR }
// 偶数行
shading: { fill: "FFFFFF", type: ShadingType.CLEAR }
```

## 设计原则

1. **留白**: 边距和间距提供呼吸空间，不要塞满页面
2. **对比**: 标题与正文之间有明显的字号/粗细对比
3. **对齐**: 统一使用一种对齐方式（左对齐/两端对齐）
4. **重复**: 同一层级的标题使用相同的格式
5. **层次**: 标题层级字号递减，不跳级（H1→H3 不允许）
6. **亲密**: 相关内容靠近，不相关内容拉开间距
